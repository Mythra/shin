# Shin #

Shin is a collection of tools I've built for collecting information related
to the game Phasmophobia. These tools are not built for any sort of Piracy, or
Cheating reasons; I also want to say here ***NEVER USE THIS WORK FOR THOSE
PURPOSES.*** If you ruin it, it doesn't just ruin it for you, but all of us.

Most tools will have varying levels of documentation, I will do my best to make
them user friendly. That being said I can't make any promises are these are
incredibly spare time projects.
